Summary
This release significantly expands MyBookmarks' data-driven workflow capabilities. It adds local file-backed integrations, richer parsing and mapping options, and a more capable prompt-flow experience while also hardening key dialogs and runtime behavior.

Changes

Features

* Added a configurable local file API service plus optional `file:///` proxy access, enabling local data sources to participate in Data Integrations.
* Added a guided Data Integrations wizard to create or extend connections, endpoints, mappings, placeholder sources, and actions with live preview and review steps. (alpha version)
* Expanded response parsing with CSV support and parser options for CSV/TSV, JSONL, YAML, HTML, lines, and text-based sources.
* Added reusable placeholder definitions and stronger expression tooling, including inline JMESPath/JSONata assistance and better path suggestions.
* Introduced major Prompt Flow upgrades, including API load steps, assign steps, workflow-variable support, branch conditions, and execution-step enhancements.

Improvements

* Renamed the API area to Data Integrations and improved its dialog layout, viewport usage, field help, path suggestion behavior, and overall editor ergonomics.
* Aligned the ref-link placeholder editor with the main link editor and improved placeholder reuse, search/replace access, and definition labeling across dialogs.
* Improved UI responsiveness with safer render invalidation, lighter Windows Chromium effects, and reduced synchronous work in key dialogs and board interactions.

Fixes

* Fixed repeated endpoint retry loops when the proxy is offline and corrected `file:///` URL joining for file-based connections.
* Fixed placeholder and dialog regressions around new-link placeholder materialization, hidden Replace values entry points, scrolling issues, and remaining EN/DE translation gaps.
* Hardened security around group HTML sanitization, redirect handling, reorder permissions, sync consistency, and save error reporting.

Documentation / Wiki Updates

* Added and refreshed setup guides for Data Integrations, including quick start, cascading selects, and Jira/GitLab examples.
* Updated the proxy and local file API documentation to cover the new local-data workflows.

Technical / Internal

* Expanded Playwright regression coverage for parsers, placeholder workflows, prompt flow, and Data Integrations behavior.
