Summary
This release expands placeholder workflows and hidden-item controls while significantly reducing UI lag in larger datasets. It also hardens favicon and proxy behavior, encrypted restore flows, and several placeholder edge cases.

Changes

Features

* Added placeholder rules with inheritance and override support, including runtime actions for visibility, required fields, option lists, and reset behavior.
* Expanded placeholder tooling with cross-scope transfer actions, reverse usage lookup, and suggestions for extracting repeated literal values into placeholders.
* Added hidden groups and links controls with optional transparency and cleaner built-in reveal styling.
* Added local Python and PowerShell HTTP proxy scripts for favicon and transport workflows.

Improvements

* Reduced UI lag during typing, drag and resize, keyword suggestions, Visual CSS filtering, and save operations, with noticeable gains on larger datasets and in Edge.
* Refined placeholder and dialog UX with draggable and resizable dialogs, a simplified prompt layout, dark-template polish, and more detailed backup activity logging with retention settings.

Fixes

* Fixed favicon rule handling so final versus fallback behavior is preserved and existing custom or proxy icons are not unintentionally reset or reloaded.
* Fixed encrypted full-export restore so imported data is persisted correctly even when security settings exist without an active local key.
* Fixed several placeholder edge cases, including stale rule references after deletion, inherited rule revalidation, case-insensitive field matching, and parsing limited to real `{{...}}` placeholders.

Documentation / Wiki Updates

* Updated wiki and in-app help for placeholder rules, global placeholders, prompt behavior, the expression editor, and the sidebar feature overview.
* Added proxy installation and usage documentation for the new local HTTP proxy scripts.

Technical / Internal

* Consolidated dialog, modal, link-template, share, and keyword helper flows, expanded Playwright coverage across the new features, and extended build packaging to produce ZIP artifacts.
