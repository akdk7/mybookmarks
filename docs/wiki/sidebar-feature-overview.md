# Sidebar: Feature Overview

## Purpose
Shows built-in feature help directly in the app (German/English).

## Content
The sidebar groups feature explanations into areas such as:
- Core features
- Collaboration and sync
- Customization
- Tools
- Keyboard
- Notes

## Newly covered feature areas
The current sidebar content now also calls out:
- Advanced link types such as separators, notes, and reference links
- Expression Editor workflows for text and URL templates
- Global placeholders with typed values and inheritance
- Placeholder rules with layered prompt behavior, overrides, validation, and link-local workflow-var conditions
- Guided link setup with simple/advanced mode, field source badges, Prompt Flow presets, and per-step controls
- API integrations with connections, endpoints, response mappings, placeholder sources, link actions, and deep shortcuts from Prompt Flow
- Reusable link templates
- Backup urgency indicator and backup activity log with concrete change summaries
- Placeholder action prompts before opening/copying/sharing links
- Local data protection via encrypted local storage
- Manual layout resizing for group height and group column width

## Language
- `de` tab for German
- `en` tab for English

## Typical beginner value
- Fast overview without reading source code.
- Helps prioritize which features matter for your workflow.

## Important relationships
- Complements technical wiki pages with user-facing guidance.
- Many points in the sidebar map conceptually to toolbar functions.
