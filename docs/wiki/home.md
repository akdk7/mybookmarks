# MyBookmarks Wiki

A standalone bookmark manager built with HTML, CSS, and JavaScript. Vue.js powers the reactive UI layer, while Bootstrap provides layout components and utility styling. The main app runs entirely inside the browser, storing its data in localStorage. No build tooling is requiredâ€”open the file in any modern browser to explore the full experience.
